# AI-logistics-optimizer
## 1.Giới thiệu dự án
AI logistics optimizer là dự án mô phỏng chia đơn hàng tự động thông minh.
Hệ thống giúp quản lý đơn hàng, shipper, khu vực giao hàng đề xuất phân đơn phù hợp dựa trên khoảng cách địa lý.

## 2. Mục tiêu
- Quản lý đanh sách đơn hàng
- Quản lý danh sách shipper 
- Theo dõi trạng thái đơn hàng
- Phân chia đơn hàng cho shipper phù hợp
- Hiển thị vị trí đơn hàng trên bản đồ
- Xây dựng dashboard thống kê cơ bản
- Từng bước áp dụng thuật toán tối ưu hoặc AI hỗ trợ chia đơn

## 3. Giao diện
Admin: 
- Dashboard
- Orders Management (Quản lý đơn hàng)
- Shippers Management
- Auto Assign(Chỉ định)/Map

User:
- Landing page
- Create Order
- Track Order
- Order History
